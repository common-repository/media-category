== Description ==

This plugin helps administrators of the site categorize their wordpress media.
If you need additional help you can contact us for [Custom Development](https://wbcomdesigns.com/contact/).

== Installation ==

1. Upload the entire "wp-media-category" folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= How can we add media category? =

Just like other post types categories, you have assign any media to any category, which you can create under the admin menu, "Media"
